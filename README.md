# BTM-update

## 初次部署 2024

### step1

```bash
curl -O https://raw.githubusercontent.com/VagrantPi/BTM-update/refs/heads/main/step1-1.sh && bash step1-1.sh
```

看到後複製後面那串 `postgres://lamassu_pg:....`，並按下 control + c 離開
